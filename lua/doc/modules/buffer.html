Buffer.input ([name=false], text, cb, opts)
Open buffer and run callback when table.keys are pressed

Parameters:

  • name Name of the scratch buffer. If skipped then create a unique id (default false)
  • text Text to display in the input buffer
  • cb Callback to run at keypress
  • opts Contains other options

Returns:

    Buffer

Usage:

    Buffer.input(name, text, cb, {
  -- Split vertically or horizontally?
  split = 's' or 'v'

  -- Comments start with ? (default: #)
  comment = '#' or string

  -- When to run callback?
  table.keys = 'gx' or string
})
