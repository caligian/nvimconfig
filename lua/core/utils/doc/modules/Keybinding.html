K:update ()

K:init (mode, lhs, cb, rest)
Create a keybinding

Parameters:

  • mode string or table If string is passed, it will be split, so that instead of passing a table, you can also pass 'nvo' for {'n', 'v', 'o'}. Any non-documented option is anything compatible with vim.keymap.set
  • lhs string Keys to map to
  • cb string or function Callback/RHS
  • rest (Optional) If string then opts.desc will be automatically set. If table then rest of the options
    • noremap Set as a non-recursive map (default: false)
    • remap Set as a recursive map (default: true)
    • leader Set as a leader map
    • localeader Set as a localeader map
    • prefix string Prefix lhs. Cannot be used with rest.localleader or rest.leader
    • pattern string or array.rest.event cannot be nil if this is used. Autocommand event(s) to bind to
    • event string or array.rest.pattern cannot be nil if this is used. Autocommand pattern(s) to bind to

Returns:

    object
