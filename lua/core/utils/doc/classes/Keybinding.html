Keybinding:init (mode, lhs, cb, rest)
Constructor

Parameters:

  • mode array[string]|string. If string is passed, it will be split, so that instead of passing a table, you can also pass 'nvo' for {'n', 'v', 'o'}. Any non-documented option is anything compatible with vim.keymap.set
  • lhs Keys to map to
  • cb string|callable
  • rest (Optional) If string then opts.desc will be automatically set. If table then rest of the options
    • noremap Set as a non-recursive map (default: false)
    • remap Set as a recursive map (default: true)
    • leader Set as a leader map
    • localeader Set as a localeader map
    • prefix Prefix lhs. Cannot be used with rest.localleader or rest.leader
    • event array[string]|string rest.pattern cannot be nil if this is used
    • pattern array[string]|string rest.event cannot be nil if this is used
    • name Unique name to reference this instance

Returns:

    object
